# Interior Design Photos

Drop inspiration photos or reference images here.

## Purpose

These photos are your visual reference for interior design decisions. Once you have photos you like:
1. Drop them here (jpg/png/webp)
2. Share them in Claude Code — Claude can read the images and help translate them into texture choices or 3D scene updates

## Suggested organization

```
design-photos/
  kitchen/
    inspo-01.jpg        ← cabinet color reference
    countertop-ref.jpg  ← quartz vs butcher block test
  bedroom/
    paint-swatch.jpg
  bathroom/
    tile-option-a.jpg
    tile-option-b.jpg
  exterior/
    siding-color.jpg
    front-door-color.jpg
```

## Workflow

1. Find a finish/color/material you like online or in a store
2. Take a photo or screenshot, drop it here
3. Tell Claude: "use the color from design-photos/kitchen/cabinet-ref.jpg for the kitchen cabinets"
4. Claude reads the image and updates the scene colors/textures
